Builds the app for production to the `build` folder.\

The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!

### `npm run eject`

**Note: this is a one-way operation. Once you `eject`, you can't go back!**

If you aren't satisfied with the build tool and configuration choices, you can `eject` at any time. This command will remove the single build dependency from your project.

Instead, it will copy all the configuration files and the transitive dependencies (webpack, Babel, ESLint, etc) right into your project so you have full control over them. All of the commands except `eject` will still work, but they will point to the copied scripts so you can tweak them. At this point you're on your own.

You don't have to ever use `eject`. The curated feature set is suitable for small and middle deployments, and you shouldn't feel obligated to use this feature. However we understand that this tool wouldn't be useful if you couldn't customize it when you are ready for it.

# UI/UX Portfolio

A responsive, high-performance portfolio website built with **React**, **Bootstrap**, and **Framer Motion**. This project showcases my UI/UX case studies with a focus on clean typography, interactive grids, and seamless navigation.

## Live Demo
[https://normaville.github.io/](https://normaville.github.io/)

## Features
- **Case Studies:** Uses React Router to load project details dynamically from a JSON data structure.
- **Modern UI/UX Design:** Inspired by minimalist design principles with high-contrast typography and intentional white space.
- **Responsive Grid:** Built with Bootstrap 5 for responsiveness accross Mobile, Tablet, and Desktop.
- **Interactive Elements:** Animations and hover effects using Framer Motion.
- **Validated Contact Form:** Built with `react-hook-form` for real-time user input validation.

## Tech Stack
- **Frontend:** React.js (Create React App)
- **Styling:** Bootstrap 5 & Custom CSS
- **Animations:** Framer Motion
- **Routing:** React Router v6
- **Forms:** React Hook Form

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate to the project folder:**
   ```bash
   cd your-repo-name
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm start
   ```
   *The site will open at `http://localhost:3000`*

## 📂 Project Structure

```text
├── components/  # Reusable UI (Navbar, Footer, ProjectCard)
├── data/        # JSON/JS files for case study content
├── pages/       # Home, About, Projects, and CaseStudy template
└── App.js       # Route configuration
```
